## Design & Planning Tools

- **GoBilda / OnShape** — For CAD design (VEX teams especially)
- **Google Slides / Canva** — For FLL Innovation Project presentations
- **Notion / Google Docs** — Team notebooks and documentation
- **GitHub** — Code versioning (you're already here!)

- **Timer apps** — Practice match timing (many FLL/VEX coach apps available)
