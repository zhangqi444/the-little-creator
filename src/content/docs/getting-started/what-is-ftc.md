FIRST Tech Challenge (FTC) is a robotics program for **grades 7–12 (ages 12–18)** in the FIRST family of programs. Teams design and build a dynamic robot from a kit of parts, then compete in a yearly themed game. FTC is the natural step up from FLL Challenge — same FIRST community, more engineering depth, longer build seasons, more student ownership.

Our wiki's FTC coverage is just getting started. Use this page as a starting point and follow the links into the [FTC Resource Map](/resources/ftc-resource-map/) for canonical sources.

## How FTC compares to FLL and FRC

| Program | Ages / Grades | Hardware | Team size | What they do |
|---|---|---|---|---|
| FLL Challenge | 9–16 / grades 4–8 | LEGO Education SPIKE Prime | 2–10 | Autonomous LEGO robot + Innovation Project |
| **FTC** | **12–18 / grades 7–12** | **REV Control Hub + goBILDA parts** | **8–12 typical** | **Hybrid autonomous + driver-controlled robot in 2.5-min matches** |
| FRC | 14–18 / grades 9–12 | Open hardware kit + custom fab | 25+ typical | Industrial-sized robot built in a 6-week season |

Most FLL Challenge kids progress to FTC between grades 7 and 9.

## A FTC season at a glance

- **September** — New game released (most recent: DECODE, 2025/26; next: BIOBUZZ, 2026/27)
- **Sept–Dec** — Build season: design robot, write autonomous + driver-control code, iterate
- **Dec–Feb** — Qualifying tournaments
- **Feb–Apr** — Regional/state championships and FIRST Championship for advancing teams

Each season has matches that combine an **autonomous period** (robot runs pre-programmed code) and a **driver-controlled period** (student drivers operate via controllers). Teams are judged on robot performance, design process, outreach, and the broader FIRST Core Values.

## What you need to get started

- **Team:** 2 lead adult coaches (required) + typically 8–12 students in grades 7–12
- **Hardware:** REV Control Hub, Driver Hub (for student drivers), goBILDA or similar build kit
- **Software:** Android-based programming environment (FTC SDK). Versions 9.0+ required. Java or Blockly.
- **Registration:** Through FIRST (firstinspires.org). Same Dashboard / Storefront flow as FLL — see the [CANOPY registration timeline](/resources/fll-resource-map/#canopy-202627-registration-timeline--new-first-dashboard--storefront) for the 2026/27 cycle.

## Programming in FTC

FTC robots run on Android-based **Control Hub** + **Driver Hub** hardware (REV Robotics). Programming environments:

- **Blockly** — visual programming, browser-based (good entry point)
- **OnBot Java** — text-based Java in browser
- **Android Studio** — full Java IDE, used by advanced teams

The official SDK is open-source on [GitHub](https://github.com/FIRST-Tech-Challenge/FtcRobotController). Full docs at [ftc-docs.firstinspires.org](https://ftc-docs.firstinspires.org/).
