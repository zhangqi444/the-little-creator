A curated map of where to go for VEX Robotics information. Every link is annotated with what it is, who it's for, and when to use it. We don't republish content — we point at it. Official sources are authoritative; everything else is listed as independent or community.

## Official sources (canonical)

### VEX Robotics — program home

- **URL:** https://www.vex.com
- **Authority:** official
- **Audience:** families, coaches, teachers, teams
- **Level:** all
- **Tags:** vex-iq, vex-v5, official, program-home
- **Use when:** You need program announcements, hardware information, curriculum resources, or the official VEX product catalog.

The main VEX Robotics site. Covers all three competition levels (IQ, V5/VRC, U) plus VEX GO for younger students. Use it to understand the product ecosystem, find curriculum resources, and access the VEX Education hub.

### VEX IQ Competition — program page

- **URL:** https://www.vexrobotics.com/viqc-current-game
- **Authority:** official
- **Audience:** coaches, teams, families
- **Level:** all
- **Tags:** vex-iq, competition, season, current-game
- **Use when:** You need the current VEX IQ season game information — field setup, game manual, scoring, and official season materials.

The official VEX IQ Competition (VIQC) current game page. Each season introduces a new game. This page links to the game manual, field specifications, scoring rules, and official season materials. The game manual is the authoritative source for competition rules — read it before designing robot strategy.

### VEX Robotics Competition (V5/VRC) — program page

- **URL:** https://www.vexrobotics.com/v5-current-game
- **Authority:** official
- **Audience:** coaches, teams (high school), families
- **Level:** intermediate, advanced
- **Tags:** vex-v5, vrc, competition, season, high-school
- **Use when:** You need current VEX V5 season information, game manual, or are preparing for the step up from VEX IQ to V5.

The VEX Robotics Competition (V5) current game page. VRC is the high school–level program (grades 7–12). Game manual, field specs, and season materials are here. If your team is transitioning from VEX IQ to V5, this is where the V5 season lives.

### RobotEvents — event finder and team registration

- **URL:** https://www.robotevents.com
- **Authority:** official
- **Audience:** coaches, teams, families
- **Level:** all
- **Tags:** vex-iq, vex-v5, events, registration, team-finder
- **Use when:** Finding local tournaments, registering your team for events, checking Skills Challenge rankings, or looking up registered teams in your region.

The official event management platform for VEX, FIRST, and other robotics programs. For VEX teams: search events by program (VIQC or VRC), date, and location; register for tournaments; view team rosters; and check Skills Challenge standings. Create a free account to register for events.

### VEXcode — official programming environment

- **URL:** https://www.vex.com/vexcode
- **Authority:** official
- **Audience:** kids, teens, coaches, teachers
- **Level:** beginner, intermediate
- **Tags:** vex-iq, vex-v5, programming, vexcode, blocks, python, cpp
- **Use when:** Downloading or accessing the programming environment for VEX IQ or V5; accessing the built-in Python and C++ API docs.

The free official programming app for all VEX hardware. Supports Blocks (drag-and-drop), Python, and C++. Available for Windows, macOS, iPadOS, Android, and Chromebook. The app includes a built-in help panel with the full Python and C++ API reference for VEX IQ and V5.

### VEX Python API reference (VEX IQ)

- **URL:** https://api.vex.com
- **Authority:** official
- **Audience:** teens, coaches
- **Level:** intermediate, advanced
- **Tags:** vex-iq, python, api, programming, reference
- **Use when:** Looking up a specific Python class, method, or parameter for VEX IQ programming — motor control, sensor readings, controller input, drivetrain methods.

The official VEX Python API documentation. Covers all VEX IQ (and V5) Python classes with method signatures, parameter descriptions, and return types. Bookmark this once your team moves past basic Blocks.

### VEX Education — curriculum resources

- **URL:** https://education.vex.com
- **Authority:** official
- **Audience:** teachers, coaches
- **Level:** beginner, intermediate
- **Tags:** vex-iq, curriculum, lesson-plans, stem, teachers
- **Use when:** Looking for structured lesson plans, STEM activities, and curriculum materials for classroom or club use.

Official VEX Education hub with curriculum for VEX GO, IQ, and V5. Includes unit plans, individual lessons, STEM labs, and coding activities. Organised by product and grade level. Most materials are free with a VEX educator account.

## Community resources (independent)

### VEX Forum

- **URL:** https://www.vexforum.com
- **Authority:** community
- **Audience:** teams, coaches, parents
- **Level:** all
- **Tags:** vex-iq, vex-v5, forum, community, q-and-a
- **Use when:** You have a specific technical question, rules clarification need, or strategic question that's not answered in official docs. Search before posting — most questions have already been answered.

The primary community forum for VEX teams worldwide. Sections for both VIQC (IQ) and VRC (V5), plus programming, building, and general discussion. Official VEX staff and experienced coaches are active here. The Rules Q&A subforum is the authoritative place for official rules clarifications during the season — answers there carry the same weight as the game manual.

### VEX IQ Subreddit — r/vexiq

- **URL:** https://www.reddit.com/r/vexiq/
- **Authority:** community
- **Audience:** teams, parents, coaches
- **Level:** all
- **Tags:** vex-iq, community, reddit, q-and-a
- **Use when:** Looking for informal advice, sharing your robot, or getting feedback from the broader VEX IQ community.

Active community for VEX IQ teams and families. More casual than the VEX Forum. Good for quick questions, robot photos, and community support. Not a source for official rules interpretations.

### VRC Subreddit — r/FTC (crossover for high school)

- **URL:** https://www.reddit.com/r/vex/
- **Authority:** community
- **Audience:** teens, coaches (high school)
- **Level:** intermediate, advanced
- **Tags:** vex-v5, vrc, community, reddit
- **Use when:** High school teams looking for V5/VRC community discussion, robot reveals, programming advice.

Community subreddit for VEX V5 / VRC teams. More technical than r/vexiq. Covers code, strategy, and competition discussion for high school teams.

### PROS — open-source C++ framework for VEX V5

- **URL:** https://pros.cs.purdue.edu
- **Authority:** independent (Purdue Robotics)
- **Audience:** advanced teens, university teams
- **Level:** advanced
- **Tags:** vex-v5, cpp, pros, open-source, programming
- **Use when:** Your V5 team wants to use C++ with an open-source toolchain instead of VEXcode — typically competitive high school or university teams optimising autonomous routines.

PROS is an open-source C++ framework for VEX V5, maintained by Purdue Robotics. Used by many top-ranked V5 teams for its flexibility and community-developed libraries (including ARMS and LemLib for autonomous movement). Not relevant for VEX IQ or beginning teams — start with VEXcode.

### LemLib — autonomous movement library for VEX V5

- **URL:** https://lemlib.readthedocs.io
- **Authority:** community
- **Audience:** advanced teens
- **Level:** advanced
- **Tags:** vex-v5, cpp, lemlib, autonomous, odometry
- **Use when:** Your V5 team uses PROS and wants a well-documented odometry and motion library for autonomous routines.

Community-maintained C++ library for PROS providing Odometry, PID, and Pure Pursuit for VEX V5 autonomous. Not relevant for VEX IQ or beginning teams.

## Progression pathway

VEX has a clear age-based progression:

| Program | Ages / Grades | Hardware | Next step |
|---|---|---|---|
| VEX GO | K–5 | Snap plastic, simple motors | VEX IQ |
| VEX IQ (VIQC) | 8–14 / grades 3–8 | Snap plastic, smart motors | VEX V5 |
| VEX V5 (VRC) | 11–18 / grades 7–12 | Metal + pneumatics | VEX U / FRC |
| VEX U | University | Full custom | — |

Most families coming from FLL move to **VEX IQ** first (ages 8–14), then optionally step to **VEX V5** in middle or high school. See [What is VEX?](/getting-started/what-is-vex/) for a fuller overview.
