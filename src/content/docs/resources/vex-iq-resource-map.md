A curated map of where to go for VEX IQ information. Every link is annotated with what it is, who it's for, and when to use it. We don't republish content — we point at it. Official sources are authoritative; everything else is listed as independent or community.

## Official sources (canonical)

### VEX IQ — program home

- **URL:** https://www.vex.com/vex-iq
- **Authority:** official
- **Audience:** families, coaches, teachers, teams
- **Level:** all
- **Tags:** vex-iq, official, program-home, hardware
- **Use when:** You need an overview of the VEX IQ system — what the hardware looks like, what age group it's designed for, and how it fits into the broader VEX ecosystem.

The main VEX IQ landing page. Covers the snap-together plastic system, smart motor / sensor ecosystem, and the VEX IQ Competition (VIQC). Good first stop for families considering VEX IQ for the first time, and for comparing the IQ platform to VEX GO (younger kids) or VEX V5 (older/more advanced).

### VEX IQ Competition (VIQC) — current game page

- **URL:** https://www.vexrobotics.com/viqc-current-game
- **Authority:** official
- **Audience:** coaches, teams, families
- **Level:** all
- **Tags:** vex-iq, viqc, competition, season, current-game
- **Use when:** You need the current VEX IQ season's game information — field setup, scoring, game manual, and official season materials. Read the game manual before designing robot strategy.

The official VIQC current game page. Each season introduces a new game. This page links to the game manual, field specifications, scoring rules, and official season materials. **The game manual is the authoritative source for competition rules** — always check the current manual when rules questions arise, as our guides may not reflect the latest season.

### VIQRC Knowledge Base (RECF)

- **URL:** https://viqrc-kb.recf.org/hc/en-us
- **Authority:** official
- **Audience:** coaches, teams, families, event partners
- **Level:** all
- **Tags:** vex-iq, viqrc, recf, kb, rules, registration, support
- **Use when:** You have specific how-to or policy questions about VEX IQ Competition — registration steps, rules interpretations, eligibility, getting-started guidance, or event partner procedures.

The official Robotics Education & Competition Foundation (RECF) knowledge base for the VEX IQ Robotics Competition (VIQRC). Covers team registration, robot eligibility, game rules clarifications (Q&A), season timeline, and event partner information. This is the canonical place for "is this allowed?" questions — authoritative in the same way the game manual is.

### RobotEvents — event finder and team registration

- **URL:** https://www.robotevents.com
- **Authority:** official
- **Audience:** coaches, teams, families
- **Level:** all
- **Tags:** vex-iq, events, registration, team-finder, skills-challenge
- **Use when:** Finding local VEX IQ tournaments, registering your team for events, checking Skills Challenge standings, or looking up registered teams in your region.

The official RECF event management platform. For VEX IQ teams: select program "VEX IQ Robotics Competition (VIQRC)", then filter by date and location. Create a free account to register for events. Skills Challenge rankings — which can influence World Championship qualification — are also tracked here.

### VEXcode — official programming environment

- **URL:** https://www.vex.com/vexcode
- **Authority:** official
- **Audience:** kids, coaches, teachers
- **Level:** beginner, intermediate
- **Tags:** vex-iq, vexcode, blocks, python, programming, download
- **Use when:** Downloading or launching the programming environment for VEX IQ; accessing built-in Python API docs.

The free official programming app for VEX IQ hardware. Supports **Blocks** (drag-and-drop for beginners) and **Python** (natural next step). Available for Windows, macOS, iPadOS, Android, and Chromebook. The built-in help panel includes the full Python API reference — you don't need a separate browser tab once you're coding.

### VEX Python API reference (VEX IQ)

- **URL:** https://api.vex.com
- **Authority:** official
- **Audience:** teams, coaches
- **Level:** intermediate, advanced
- **Tags:** vex-iq, python, api, reference, programming
- **Use when:** Looking up a specific VEX IQ Python class, method, or parameter — motor control, sensor readings, controller input, drivetrain methods.

The official VEX Python API documentation for VEX IQ (and V5). Covers all Python classes with method signatures, parameter descriptions, and return types. Bookmark this when your team transitions from Blocks to Python — it answers almost every "how do I read the distance sensor?" question faster than searching a forum.

### VEX Education — curriculum resources

- **URL:** https://education.vex.com
- **Authority:** official
- **Audience:** teachers, coaches
- **Level:** beginner, intermediate
- **Tags:** vex-iq, curriculum, lesson-plans, stem, classroom
- **Use when:** Looking for structured lesson plans, STEM activities, and curriculum materials for a classroom or after-school club.

Official VEX Education hub with curriculum for VEX GO, IQ, and V5. Includes unit plans, individual lessons, STEM labs, and coding activities organised by product and grade level. Most materials are free with a VEX educator account. Particularly useful for classroom coaches who want pre-built activities rather than designing from scratch.

## REC Foundation resources

VEX hardware is made by VEX Robotics; VEX *competitions* are run by the **Robotics Education & Competition Foundation (RECF)**, a separate non-profit. Many first-year families miss the RECF side entirely — and miss out on grants, scholarships, mentor training, and rules Q&A resources.

### REC Foundation — program operator

- **URL:** https://recf.org
- **Authority:** official
- **Audience:** families, coaches, teachers, event partners
- **Level:** all
- **Tags:** recf, vex-iq, governance, official, grants, scholarships
- **Use when:** Understanding who runs VEX competitions, finding cross-program policies, or accessing resources like grants and scholarships.

RECF runs all VEX competition programs — VIQRC (IQ), V5RC (V5), VURC (University), VAIRC (AI) — plus the Aerial Drone Competition. RECF handles team registration policies, event partner accreditation, rules processes, mentor training, and the World Championship. If you're an FLL family: think of RECF as the equivalent of FIRST HQ.

### New to Robotics — RECF landing page

- **URL:** https://recf.org/new-to-robotics/
- **Authority:** official
- **Audience:** families, parents, new teachers
- **Level:** beginner
- **Tags:** recf, getting-started, new-families, vex-iq
- **Use when:** You're brand-new to VEX and need a structured orientation from the program operator.

RECF's curated entry point for families with no prior robotics exposure. Walks through the different VEX programs, how a season works, and how to find or start a team. Complements our [What is VEX?](/getting-started/what-is-vex/) overview — that's the narrative; this is the official structure.

### RECF Trainings — coach and event partner courses

- **URL:** https://recf.org/recf-trainings/
- **Authority:** official
- **Audience:** coaches, teachers, event partners, volunteers
- **Level:** beginner, intermediate
- **Tags:** recf, coach-training, professional-development
- **Use when:** You're a new VEX IQ coach who wants structured RECF training on running a team or volunteering at events.

Free RECF training tracks for adults supporting VEX teams. Covers the "new coach" role, event partner onboarding, and seasonal game updates. The "new coach" track is useful even if you'll never be an event partner — it explains how the season works from the inside.

### Team Grants — RECF

- **URL:** https://recf.org/teams/for-participants/grants/
- **Authority:** official
- **Audience:** new coaches, teachers, parents starting a team
- **Level:** beginner
- **Tags:** recf, grants, funding, equipment, vex-iq
- **Use when:** You're starting a new VEX IQ team and need help with the hardware investment.

RECF runs a Team Grant Program matching schools and community organisations with sponsor-funded resources — typically a free or discounted starter kit. Eligibility varies by sponsor (some grants are region-specific; some target Title I schools or underserved students). Apply early. See also our [Team Funding guide](/guides/team-funding/) for broader fundraising ideas.

### Girl Powered — RECF initiative

- **URL:** https://recf.org/teams/for-participants/girl-powered/
- **Authority:** official
- **Audience:** coaches, parents, all-girls and co-ed teams
- **Level:** all
- **Tags:** recf, girl-powered, diversity, inclusion
- **Use when:** You coach or want to recruit girls into VEX IQ, or you want to host an outreach workshop.

RECF's Girl Powered initiative (presented by Google) funds workshops, awards, and outreach materials for getting and keeping more girls in robotics. Includes a workshop kit, a separate Girl Powered Award at most tournaments, and a scholarship track. Useful even for co-ed teams — the inclusion principles apply across the board.

## Season & events

### VEX Robotics World Championship

- **URL:** https://recf.org/vex-robotics-world-championship/
- **Authority:** official
- **Audience:** qualifying teams, coaches, families planning long-term
- **Level:** advanced
- **Tags:** vex-iq, worlds, championship, qualification
- **Use when:** Your team is approaching regional qualification, or you want to understand what the top of VEX IQ looks like.

The annual culmination of all VEX programs, held in late April (2026 edition: St. Louis, April 21–30). VEX IQ teams qualify through regional and state events — check the [VIQRC KB](https://viqrc-kb.recf.org/hc/en-us) for your program's specific qualification pathway. Even if your team isn't competing, watching the live stream is motivating for the whole team.

### RECF Signature Events

- **URL:** https://recf.org/teams/competition/recf-signature-events/
- **Authority:** official
- **Audience:** competitive teams, coaches
- **Level:** intermediate, advanced
- **Tags:** recf, signature-events, regional
- **Use when:** Your team wants a step up from local tournaments — Signature Events draw multi-region teams and use official branded fields.

RECF-designated regional tournaments larger than typical local events (often 60–80+ teams). A good intermediate goal between local qualifiers and the World Championship.

## Community resources (independent)

### VEX Forum

- **URL:** https://www.vexforum.com
- **Authority:** community
- **Audience:** teams, coaches, parents
- **Level:** all
- **Tags:** vex-iq, community, forum, q-and-a, rules
- **Use when:** You have a specific technical, build, or rules question that isn't answered in official docs. Search before posting — most questions are already answered.

The primary community forum for VEX teams worldwide. Includes dedicated sections for VEX IQ (VIQC) and VRC, plus programming and building. Official VEX staff are active. The Rules Q&A subforum contains official rules clarifications that carry the same weight as the game manual.

- **Authority:** community
- **Audience:** teams, parents, coaches
- **Level:** all
- **Tags:** vex-iq, community, reddit, q-and-a

Casual community for VEX IQ teams and families. More relaxed than the VEX Forum. Good for quick questions, robot photos, build feedback, and hearing from other first-year coaches.
