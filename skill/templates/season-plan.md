## Phase 1 — Kickoff (August–September)

**Goal:** Understand the challenge, form the team, set up equipment.

- [ ] Register on FIRST Dashboard (my.firstinspires.org)
- [ ] Register with regional PDO
- [ ] YPP screening completed for all adult members
- [ ] Watch season reveal video together as a team
- [ ] Read Robot Game rulebook — highlight key missions
- [ ] Unbox and test SPIKE Prime set
- [ ] Assign team roles (see team charter)
- [ ] Set season goals together
- [ ] Start engineering notebook — first entry: team goals and first impressions of the challenge

**Innovation Project:**
- [ ] Identify season theme topic area
- [ ] Brainstorm 5+ real-world problems related to the theme
- [ ] Pick one problem to research — document the decision

---

## Phase 2 — Build season (September–October)

**Goal:** Working robot, shortlisted missions, IP research underway.

**Robot Game:**
- [ ] Score all missions — build expected-value table
- [ ] Choose 8–10 missions to focus on
- [ ] Design and build attachment(s) for priority missions
- [ ] Write first autonomous programs for each mission
- [ ] Run each mission 10 times — track success rate
- [ ] Identify 2–3 "reliable" missions (>80% success rate)

**Innovation Project:**
- [ ] Research the identified problem (minimum 3 sources)
- [ ] Interview or consult at least one expert or real stakeholder
- [ ] Draft an innovative solution — specific, testable idea
- [ ] Begin presentation structure (problem → research → solution → sharing plan)

**Notebook:** Weekly entries minimum. Every design decision documented.

---

## Phase 3 — Polish (October–November)

**Goal:** Consistent robot scores, polished IP, mock judging done.

**Robot Game:**
- [ ] All priority missions running >90% in practice
- [ ] Run order finalized — fastest/most reliable sequence
- [ ] Attachment quick-changes practiced (if using multiple)
- [ ] Backup strategies for common failures documented

**Innovation Project:**
- [ ] Presentation rehearsed (5 minutes, all members speak)
- [ ] Visual aid / poster / slides complete
- [ ] Solution tested or piloted in some way (even a paper prototype counts)
- [ ] Shared with at least one real person outside the team

**Core Values:**
- [ ] Mock Core Values judging session (with parent or guest judge)
- [ ] Every member can answer: "What's your team's best Core Values example?"
- [ ] GP moment from this season identified and practiced

**Robot Design:**
- [ ] Mock Robot Design judging session
- [ ] Every member can explain one design decision and the reasoning behind it

---

## Phase 4 — Tournament day

**Goal:** Execute what you've practiced. Stay calm. Have fun.

- [ ] Pack list complete (see [Tournament Day Checklist](https://zhangqi444.github.io/the-little-creator/guides/tournament-day-checklist/))
- [ ] Robot tested morning of tournament
- [ ] Judging schedule reviewed — know your room and time
- [ ] Team fed and rested
- [ ] Coach role: logistics, not coaching during runs
