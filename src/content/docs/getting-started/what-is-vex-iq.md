VEX IQ is a competitive robotics program for **students ages 8–14 (elementary through middle school)**, run by the [Robotics Education & Competition Foundation (RECF)](https://www.recf.org/). Teams build robots from a snap-together plastic kit — no soldering, no power tools — then compete in a yearly game on a field roughly the size of a large dining table.

:::note[Current season]
The 2025-26 VEX IQ season is **Push Back** (VIQRC). The 2026-27 season, **Override**, was announced for transition in summer 2026. Check the [VIQRC current game page](https://www.vexrobotics.com/viqc-current-game) for the latest season materials and official game manual.
:::

## Who is VEX IQ for?

VEX IQ is designed for students roughly in the 3rd–8th grade range. It's a natural step up from LEGO-based programs (like [FLL](/getting-started/what-is-fll/)) for students who want more mechanical depth and a longer competitive season. It's also a popular on-ramp for students who haven't done competitive robotics before and want to enter the VEX ecosystem.

Because VEX IQ uses a snap-together plastic system similar in spirit to LEGO, younger students can participate meaningfully — a motivated 9-year-old can build and program a competitive robot. By middle school, teams are typically writing Python programs, designing custom drive bases, and strategising around match play.

## How VEX IQ compares to nearby programs

| Program | Ages | Hardware | Key feature |
|---|---|---|---|
| FLL Challenge | 9–16 (grades 4–8) | LEGO SPIKE Prime | Autonomous LEGO robot + Innovation Project presentation |
| **VEX IQ** | **8–14 (elem–middle)** | **VEX IQ snap-together plastic** | **Alliance matches + Skills Challenge; no research project** |
| VEX V5 (VRC) | 11–18 (grades 6–12) | VEX V5 metal components | More complex builds; high-school level engineering |

For most FLL families, VEX IQ is the natural VEX entry point. For students moving up in the VEX pathway, VEX V5 follows. See the [Progression Guide](/getting-started/progression-guide/) for a structured look at when and how to move between programs.

## A VEX IQ season at a glance

- **Summer** — New game announced for the upcoming season
- **September–November** — Build and programming season; teams design their robot and iterate
- **December–February** — Qualifying tournaments (regional events via [RobotEvents.com](https://www.robotevents.com))
- **March–April** — State/regional championships and invitations to the VEX Robotics World Championship (held in late April)

The current 2025-26 game is **Push Back**. Planning for 2026-27? That season launches as **Override** — check the official VIQRC page for the game manual once it publishes.

## How competition works

Each VEX IQ event has two formats running in parallel:

### Alliance (Teamwork) matches

Two teams collaborate as an alliance on the same field. You are playing *with* another team, not against one — both robots work together to maximise the alliance score. This cooperative format is a defining feature of VEX IQ (VEX V5 and VEX U use head-to-head alliance competition, so this is a meaningful difference).

Qualification matches pair you with different partners each round. Alliance selection for elimination rounds is based on qualification ranking.

### Skills Challenge

Each team runs the field solo — no partner robot — in two modes:

- **Driver Skills:** operator-controlled run, 60 seconds
- **Autonomous Coding Skills:** fully autonomous run, 60 seconds (robot runs its program with no human input)

Skills scores accumulate across events throughout the season and contribute to a global ranking. Strong Programming Skills performance is often what separates regionally competitive teams from those who qualify to Worlds. For more on what Skills runs look like in practice, see the [VEX IQ Tournament Guide](/guides/vex-iq-tournament/).

## Awards

VEX IQ tournaments award teams beyond match results:

- **Excellence Award** — the top award; spans robot design, programming, skills scores, and team conduct
- **Design, Build, Teamwork, Think, Innovate, Amaze** — component awards recognising specific strengths
- **Robot Skills Champion** — highest combined Driver + Programming Skills score at the event

For a full breakdown of what each award looks for, see the [VEX Awards guide](/guides/vex-awards/).

## Programming in VEX IQ

VEX IQ uses the **VEX IQ Brain** (second-generation hardware as of the current product line), which teams program with the free **VEXcode IQ** app. Three programming options:

- **VEXcode IQ Blocks** — Drag-and-drop block programming; great entry point for ages 8–11
- **VEXcode IQ Python** — Text-based Python; most common at the competitive level for 5th grade and up
- **VEXcode IQ C++** — Available for advanced teams who want it

Most teams start with Blocks and move to Python once they're comfortable with programming concepts. The transition is straightforward — VEXcode IQ shows blocks and Python side by side. See the [VEX IQ Programming Guide](/guides/vex-iq-programming/) for practical examples covering drive base, sensors, and autonomous patterns.

## What you need to get started

- **Team:** 2 adult coaches + typically 3–6 students (RECF allows up to 10)
- **Hardware:** VEX IQ kit (V2 system) — smart motors, sensors, and the IQ Brain. Starter kits available from VEX or authorised resellers; cost is significantly lower than VEX V5 or FRC
- **Software:** VEXcode IQ (free download) for Windows, macOS, iPadOS, Android, or Chromebook
- **Registration:** Through [RobotEvents.com](https://www.robotevents.com) — register the team and find local events

## Key differences from FLL

- **No research project.** VEX IQ is purely robot competition — no Innovation Project or presentation judging (though notebook-keeping is encouraged for awards like Excellence and Design).
- **Cooperative alliance matches.** VEX IQ teams work *with* their partner in matches. FLL is solo-robot with a team presentation.
- **Skills Challenge.** FLL has no equivalent to the autonomous skills run and global skills ranking.
- **Longer season, more events.** Multiple qualifying events are typically available in a region before the state championship.
- **Metal-free.** VEX IQ uses snap-together plastic — no power tools, no metal cutting — making it more accessible for younger builders.
