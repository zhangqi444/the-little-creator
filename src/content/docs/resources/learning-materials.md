## Programming fundamentals — younger kids (ages 8–12)

### Code.org

- **URL:** https://code.org
- **Authority:** independent
- **Audience:** kids, families
- **Level:** beginner
- **Tags:** programming, block-based, free
- **Use when:** A child has never coded before and you want a friendly, browser-based first step.

Free, browser-based intro to coding with themed courses (Minecraft, Frozen, Star Wars). Block-based, runs on any device, no install. Great for ages 8–12 in their first weeks. Not robotics-specific, but the logic — sequencing, loops, conditionals — is identical to what FLL and VEX IQ programming uses.

### Scratch (MIT)

- **URL:** https://scratch.mit.edu
- **Authority:** independent
- **Audience:** kids, families
- **Level:** beginner
- **Tags:** programming, block-based, free
- **Use when:** A child wants to make games or animations while learning the same logic FLL and VEX IQ use.

Free, browser-based, block-based programming environment from MIT. Same drag-and-drop pattern as the LEGO SPIKE App and VEXcode IQ Blocks, so practice here transfers directly to FLL and VEX IQ programming. Strong online community of kid-made projects.

### LEGO Education lessons

- **URL:** https://education.lego.com
- **Authority:** official
- **Audience:** kids, coaches, teachers
- **Level:** beginner
- **Tags:** fll, spike-prime, lesson-plans
- **Use when:** You have a SPIKE Prime kit and want structured lessons before tackling competition missions.

Official LEGO Education lesson plans bundled with SPIKE Prime kits. Hands-on, classroom-tested, free with kit purchase. Good way to learn the hardware before the season's challenge drops.

## Programming fundamentals — older kids (ages 12+)

### CS50 (Harvard, free)

- **URL:** https://cs50.harvard.edu
- **Authority:** independent
- **Audience:** kids, teens
- **Level:** intermediate
- **Tags:** programming, computer-science, free, university
- **Use when:** A teen is past block-based coding and ready for real computer science fundamentals.

Harvard's free intro to computer science. Covers C, Python, SQL, web. Demanding but legendarily well-taught. A strong post-FLL next step for students hooked on programming; also relevant for FRC students wanting a CS foundation before tackling Java.

### Python.org Beginner's Guide

- **URL:** https://wiki.python.org/moin/BeginnersGuide
- **Authority:** official
- **Audience:** kids, teens, coaches
- **Level:** beginner
- **Tags:** python, free, reference
- **Use when:** A kid is moving from block-based to Python on SPIKE Prime or VEX, and needs the canonical Python tutorials.

Official Python.org curated list of beginner tutorials and books. Authoritative. Worth bookmarking when a kid graduates from blocks to text-based code.

### VEX Robotics Curriculum

- **URL:** https://education.vex.com
- **Authority:** official
- **Audience:** kids, coaches, teachers
- **Level:** beginner, intermediate
- **Tags:** vex, curriculum, free, lesson-plans
- **Use when:** A VEX team or class needs structured lessons covering build + code + game strategy.

Official VEX Education curriculum, free. Covers VEX IQ and V5 with lesson plans, videos, and assessments. Designed for classrooms but works for home teams too.

## FTC-specific learning

### FTC SDK Documentation

- **URL:** https://ftc-docs.firstinspires.org
- **Authority:** official
- **Audience:** teens, coaches
- **Level:** beginner, intermediate
- **Tags:** ftc, java, sdk, programming, official
- **Use when:** You're starting FTC programming and need the authoritative reference for FTC Blocks, OnBot Java, and Android Studio.

The official FTC documentation site. Covers all three programming environments: FTC Blocks (drag-and-drop), OnBot Java (browser-based Java), and Android Studio (full IDE). Includes hardware map setup, motor configuration, op mode structure, and common troubleshooting. First stop for any FTC programming question.

### FTC SDK on GitHub

- **URL:** https://github.com/FIRST-Tech-Challenge/FtcRobotController
- **Authority:** official
- **Audience:** teens, coaches
- **Level:** intermediate
- **Tags:** ftc, java, sdk, open-source
- **Use when:** Setting up Android Studio for FTC or reviewing example op modes from the official starter repo.

The official FTC robot controller SDK. Fork this repo to start a new FTC season. Contains example op modes for mecanum drive, encoder use, gyro stabilisation, and more. Reading the example code is one of the fastest ways to learn FTC patterns.

### Game Manual 0 — FTC Edition

- **URL:** https://gm0.org
- **Authority:** community
- **Audience:** teens, coaches
- **Level:** beginner, intermediate, advanced
- **Tags:** ftc, strategy, hardware, programming, community
- **Use when:** You need practical, opinionated guidance on FTC robot design, hardware selection, and programming that goes beyond the official docs.

Community-written reference guide covering FTC hardware choices, drive trains, mechanisms, and software patterns. Extremely well-regarded in the FTC community. Complements the official docs with real-world team experience.

### REV Hardware Client

- **URL:** https://docs.revrobotics.com/rev-hardware-client
- **Authority:** official
- **Audience:** teens, coaches
- **Level:** beginner
- **Tags:** ftc, hardware, rev-robotics
- **Use when:** Setting up REV Control Hub / Expansion Hub firmware or updating motor firmware before a season.

Official REV Robotics software for configuring and updating FTC control hardware. Essential first step for any team using REV hardware (which is the majority of FTC teams).

## FRC-specific learning

### WPILib Documentation

- **URL:** https://docs.wpilib.org
- **Authority:** official
- **Audience:** teens, coaches
- **Level:** beginner, intermediate, advanced
- **Tags:** frc, java, python, wpilib, programming, official
- **Use when:** Learning FRC programming from scratch, or looking up any WPILib class, subsystem pattern, or command-based framework question.

The primary FRC programming reference. Covers Java and Python (RobotPy). Topics range from "zero to driving a robot" (TimedRobot quickstart) through Command-based architecture, PID controllers, trajectories, and Shuffleboard dashboards. Kept up to date each season by the WPILib team.

### FRC Zero to Robot

- **URL:** https://docs.wpilib.org/en/stable/docs/zero-to-robot/introduction.html
- **Authority:** official
- **Audience:** teens, coaches
- **Level:** beginner
- **Tags:** frc, wpilib, getting-started, official
- **Use when:** A brand-new FRC team needs a guided path from robot unboxing through first autonomous.

WPILib's structured onboarding path for rookie FRC teams. Walks through tools installation, roboRIO imaging, hardware wiring, and a first driving program. Recommended as the first reading for any FRC programming mentor.

### Chief Delphi

- **URL:** https://www.chiefdelphi.com
- **Authority:** community
- **Audience:** teens, coaches
- **Level:** all
- **Tags:** frc, community, forum, strategy
- **Use when:** Searching for FRC strategy discussions, control system issues, or community advice from experienced teams.

The central FRC community forum, running since 2001. Invaluable for troubleshooting, strategy Q&A, and tracking what top teams are thinking. Has a searchable archive of years of FRC knowledge. Use the search before posting — most questions have been asked before.

### PathPlanner

- **URL:** https://pathplanner.dev/home.html
- **Authority:** community
- **Audience:** teens, coaches
- **Level:** intermediate, advanced
- **Tags:** frc, autonomous, path-planning, wpilib
- **Use when:** Planning complex FRC autonomous routines with curved paths and dynamic event handling.

Popular open-source path planning tool for FRC. GUI-based path editor with WPILib integration. Supports holonomic (swerve) and differential drivetrains. Widely used in FRC auto programming.

### RobotPy (Python for FRC)

- **URL:** https://robotpy.readthedocs.io
- **Authority:** community
- **Audience:** teens, coaches
- **Level:** intermediate
- **Tags:** frc, python, wpilib, robotpy
- **Use when:** Your FRC team prefers Python over Java and needs the RobotPy/WPILib Python bindings reference.

Community-maintained Python bindings for WPILib. Lets FRC teams write robot code in Python instead of Java or C++. Full feature parity with the Java WPILib API. A good option for teams with a strong Python background.

## VEX-specific learning

### VEX IQ Knowledge Base

- **URL:** https://kb.vex.com/hc/en-us/categories/360001134851-VEX-IQ
- **Authority:** official
- **Audience:** kids, coaches, teachers
- **Level:** beginner, intermediate
- **Tags:** vex-iq, hardware, software, official
- **Use when:** Troubleshooting VEX IQ hardware, looking up VEXcode IQ Blocks features, or finding official build guides.

Official VEX IQ knowledge base. Covers IQ Brain setup, motor/sensor configuration, VEXcode IQ Blocks and Python, and common competition questions. Start here before the VEX Forum for hardware and software questions.

### VEX V5 / VRC Knowledge Base

- **URL:** https://kb.vex.com/hc/en-us/categories/360001222332-VEX-V5
- **Authority:** official
- **Audience:** teens, coaches
- **Level:** beginner, intermediate, advanced
- **Tags:** vex-v5, vrc, hardware, software, official
- **Use when:** Looking up V5 Brain setup, Smart Motor configuration, VEXcode Pro V5, or V5RC competition rules clarifications.

Official VEX V5 knowledge base. Comprehensive coverage of V5 hardware, VEXcode Pro V5 (C++ and Python), firmware updates, and competition preparation. More technical depth than the IQ KB, reflecting the older audience.

### PROS Documentation

- **URL:** https://pros.cs.purdue.edu/v5/index.html
- **Authority:** community
- **Audience:** teens, coaches
- **Level:** intermediate, advanced
- **Tags:** vex-v5, vrc, pros, c-plus-plus, open-source
- **Use when:** Your VEX V5 team wants to program in C++ with an open-source RTOS instead of VEXcode.

Open-source C++ framework for VEX V5/VRC and VEX U, maintained by Purdue University. Supports RTOS tasks, OkapiLib motion control, and advanced autonomous programming. Standard choice for competitive VRC and VEX U teams.

### VEX Forum

- **URL:** https://www.vexforum.com
- **Authority:** community
- **Audience:** families, coaches, teens
- **Level:** all
- **Tags:** vex, community, forum
- **Use when:** Searching for VEX IQ, V5, or VEX U rule clarifications, robot design ideas, or community advice.

The central VEX community forum. Covers all VEX programs. Hosts official rule Q&As from RECF game designers. Invaluable for rules clarifications and for learning from experienced teams.

## Books

### *Invent Your Own Computer Games with Python* — Al Sweigart

- **URL:** https://inventwithpython.com/invent4thed/
- **Authority:** independent
- **Audience:** kids, teens
- **Level:** beginner
- **Tags:** python, books, free
- **Use when:** A kid wants to learn Python by building things they care about — games — instead of abstract exercises.

Free online (also sold in print). Project-based Python intro. Good companion when a kid moves from block-based FLL programming to Python on SPIKE Prime, or a VEX IQ student starts Python.

### *Python Crash Course* — Eric Matthes

- **URL:** https://ehmatthes.github.io/pcc_3e/
- **Authority:** independent
- **Audience:** teens, parents
- **Level:** beginner, intermediate
- **Tags:** python, books
- **Use when:** A motivated kid (or parent learning alongside them) wants a thorough Python foundation.

Widely used as a self-teach Python book. Project chapters at the end. Good for parents who want to keep up with their kid's programming, or FLL/VEX students making the transition to text-based code.

### *Introduction to Java Programming* — Y. Daniel Liang

- **URL:** https://www.pearson.com/en-us/subject-catalog/p/introduction-to-java-programming-and-data-structures/P200000003418
- **Authority:** independent
- **Audience:** teens, coaches
- **Level:** intermediate
- **Tags:** java, books, ftc, frc
- **Use when:** An FTC or FRC student needs a systematic Java foundation beyond what YouTube tutorials provide.

Standard university-level Java textbook. More reference than tutorial — best used alongside hands-on FTC/FRC programming rather than as a first book. Good for FTC students using Android Studio or FRC teams wanting CS depth.

## FLL-specific essentials

### Season Challenge & Game Manuals

- **URL:** https://www.firstlegoleague.org/season
- **Authority:** official
- **Audience:** coaches, teams, judges
- **Level:** all
- **Tags:** fll, rules, official, must-read
- **Use when:** Always — the game manual is the most important document of the season. Read it before designing strategy.

Each August FIRST releases the season's Challenge and the Robot Game Rule Book. These are the only authoritative source for missions, point values, penalties, and Core Values rubric. Anything inconsistent with the manual loses to the manual.
