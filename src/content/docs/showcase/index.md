This is where our kids' work lives. Robot builds, programs, competition highlights, and project writeups — all shared with pride.

## How to Add Your Work

We welcome submissions from any family in the community. To showcase your project:

1. Fork this repository
2. Create a folder under `src/content/docs/showcase/` named `[year]-[team-name]/` (e.g., `2025-team-spark/`)
3. Add an `index.md` describing your project (see the template below)
4. Include photos, videos (linked from YouTube), or code snippets
5. Submit a pull request — we'll review and merge it

## Showcase Template

````markdown
---
title: Team Name — Season/Year
description: Short description of your team and project
---

**Competition:** FLL / VEX IQ / VEX V5
**Season Theme:** e.g., "MASTERPIECE℠"
**Members:** Number of kids, ages optional

## Our Robot

Describe your robot — what it does, what you're proud of, what challenged you.

## Code Highlights

Share an interesting snippet or link to your GitHub repo.

## Innovation Project (FLL only)

Describe your research topic and solution.

## What We Learned

The most valuable thing this season taught your team.

## Photos & Videos

Link to YouTube, Google Photos album, etc.
````
